# SF6 Scouter

**SF6 Scouter** 是一款轻量级的《街头霸王6》实时战绩追踪工具。专为玩家和主播设计，实时同步胜率、排位分增减等核心数据。

> **悬浮看分，手热吃分，手冷下机。**

[English](./README.md) | [简体中文] | [日本語](./README_ja.md)

## 🚀 核心功能

- **实时追踪**：自动记录当前 Session 的胜、负场次及胜率。
- **排位分追踪 (LP/MR)**：同时支持 LP（段位分）和 MR（大师分）。清晰显示自启动程序以来的分数增减情况。
- **多账号关注**：登录后不仅可以追踪自己，还可以通过输入 **CFN ID** 追踪其他玩家的实时战绩。
- **Pro 级数据面板**：全新设计的专业仪表盘，以**折线图**形式直观展示历史战斗记录和分数波动趋势。
- **个性化记忆**：应用会自动记录您关闭时的面板配置，包括窗口位置、缩放大小、透明度等，下次启动完美复原。
- **UI 特效**：新增动态数值滚动及 UI 交互特效，让数据展现更具动感。
- **主播友好**：
  - **透明模式**：开启后可方便地通过 OBS 等软件滤镜叠加到直播画面中。
  - **窗口置顶**：即使在窗口/无边框模式下游戏，也能随时查看实时战绩。
- **自动更新**：内置更新模块，确保您第一时间获得新功能。

## 🛠️ 使用方法

1. **启动与登录**：打开 `SF6 Scouter`，点击 "Login to CFN" 并在弹出窗口中登录。
2. **账号选择**：登录成功后，默认追踪您的账号。您也可以在设置或搜索栏中输入他人的 **CFN ID** 进行跨账号追踪。
3. **切换面板**：点击界面上的切换按钮，可在“Mini 模式”与“Pro 模式”间自由切换。Pro 面板支持折线图缩放、历史记录筛选等多种自定义设置。
4. **自定义外观**：通过拖拽边缘调整大小，或在设置中调节透明度。Pro 面板的布局和显示内容也可根据需求在设置中进一步微调。所有修改将自动保存。
5. **实时监控**：应用每 15 秒会自动轮询数据。开始您的街霸之旅吧！

## 📺 OBS 配置教程

若您希望在直播中使用 **SF6 Scouter**，请按照以下步骤操作：

1. **添加来源**：
   - 在 OBS 的“来源”面板中点击 `+` 号。
   - 选择 **窗口采集 (Window Capture)**。
   - 在弹出的窗口中，将“窗口”选择为 `SF6 Scouter`。
   - 将“采集方式”设置为 **Windows 10 (1903 及以上)**。
2. **应用设置**：
   - 点击应用中的 ⚙️ 图标进入设置。
   - 将 **开启最上层显示 (Always on Top)** 切换为 **关闭 (Off)**。
