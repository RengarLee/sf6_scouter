# SF6 Scouter

**SF6 Scouter** は、ストリートファイター6 向けの軽量なリアルタイム戦績トラッカーです。プレイヤーや配信者向けに設計されており、勝敗数やランクポイントの変動をライブで表示します。

> **オーバーレイで戦績確認、勝てる時に稼ぎ、冷めたら即ヤメ。**

[English](./README.md) | [简体中文](./README_zh.md) | [日本語]

## 🚀 主な機能

- **リアルタイムトラッキング**: セッション中の勝敗数、勝率を自動で更新します。
- **ランクポイント (LP/MR)**: リーグポイント (LP) とマスターレート (MR) の両方に対応。アプリ起動時からの増減値をひと目で確認できます。
- **配信者向け機能**:
  - **透過モード**: 背景を透明にして、配信画面に直接重ねることができます。
  - **最前面表示**: ゲームプレイ中もトラッカーを常に手前に表示できます。
- **簡単セットアップ**: 公式サイト Buckler's Booty に一度ログインするだけで、アプリが自动的に CFN ID を検出します。

## 🛠️ 使い方

1. **起動**: `SF6 Scouter` を开きます。
2. **ログイン**: 「Login to CFN」ボタンをクリックします。别ウィンドウでストリートファイター6の公式サイト（Buckler's Booty）が开きます。
3. **自动検出**: ログインが完了すると、アプリが自动的にユーザーコードを検出し、トラッキングを开始します。
4. **プレイ**: アプリを启动したままゲームをプレイしてください。15 秒ごとに自动でデータを取得します。
5. **設定**: ⚙️ アイコンをクリックして以下の操作が可能です：
   - LP と MR の表示切り替え。
   - 透過モードの切り替え。
   - セッションデータのリセット。

## 📺 OBS 設定チュートリアル

配信で **SF6 Scouter** を使用する場合は、以下の手順に従ってください。

1. **ソースの追加**:
   - OBS の「ソース」パネルで `+` をクリックします。
   - **ウィンドウキャプチャ** を選択します。
   - 「ウィンドウ」で `SF6 Scouter` を選択します。
   - 「キャプチャ方式」を **Windows 10 (1903 以降)** に設定します。
2. **アプリの設定**:
   - アプリの ⚙️ アイコンをクリックして設定を开きます。
   - **最前面表示 (Always on Top)** を **オフ (Off)** に切り替えます。
