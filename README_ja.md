# SF6 Scouter

**SF6 Scouter** は、ストリートファイター6 向けの軽量なリアルタイム戦績トラッカーです。プレイヤーや配信者向けに設計されており、勝敗数やランクポイントの変動をライブで表示します。

> **オーバーレイで戦績確認、勝てる時に稼ぎ、冷めたら即ヤメ。**

[English](./README.md) | [简体中文](./README_zh.md) | [日本語]

## 🚀 主な機能

- **リアルタイムトラッキング**: セッション中の勝敗数、勝率を自動で更新します。
- **ランクポイント (LP/MR)**: リーグポイント (LP) とマスターレート (MR) の両方に対応。アプリ起動時からの増減値をひと目で確認できます。
- **配信者向け機能**:
  - **グリーンバックモード**: OBS などの配信ソフトで簡単にオーバーレイとして利用可能です。
  - **最前面表示**: ゲームプレイ中もトラッカーを常に手前に表示できます。
- **簡単セットアップ**: 公式サイト Buckler's Booty に一度ログインするだけで、アプリが自動的に CFN ID を検出します。

## 🛠️ 使い方

1. **起動**: `SF6 Scouter` を開きます。
2. **ログイン**: 「Login to CFN」ボタンをクリックします。別ウィンドウでストリートファイター6の公式サイト（Buckler's Booty）が開きます。
3. **自動検出**: ログインが完了すると、アプリが自動的にユーザーコードを検出し、トラッキングを開始します。
4. **プレイ**: アプリを起動したままゲームをプレイしてください。15 秒ごとに自動でデータを取得します。
5. **設定**: ⚙️ アイコンをクリックして以下の操作が可能です：
   - LP と MR の表示切り替え。
   - グリーンバックモードの切り替え。
   - セッションデータのリセット。
