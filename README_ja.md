# <img src="./images/logo.png" alt="SF6 Scouter Logo" width="40" valign="middle"> SF6 Scouter

**SF6 Scouter** は、ストリートファイター6のプレイヤーや配信者向けの実行可能アプリで、**リアルタイム戦績追跡**、**スコア変動表示**、**Pro オーバーレイ表示**、**データ分析** に重点を置いています。

## リポジトリについて

このリポジトリは、**実行可能な SF6 Scouter アプリの配布用リポジトリ**です。Web ページ、プレビュー画像、リリース情報、ダウンロード導線を提供します。

- ここには **アプリのソースコードは含まれていません**。
- SF6 Scouter を利用したい場合は、ビルド可能なソースを探すのではなく、リリース情報とダウンロード内容を確認してください。

## 主な機能

- **リアルタイム対戦追跡**: 現在のセッション中の勝敗、勝率、スコア変動を追跡します。
- **LP / MR 両対応**: リーグポイントとマスターレートの両方を分かりやすく表示します。
- **CFN アカウント追跡**: 一度ログインすれば、自分のアカウントだけでなく他の CFN ID にも切り替えられます。
- **Mini オーバーレイ**: プレイ中でも見やすいコンパクトな表示を提供します。
- **Pro パネル**: より情報量の多いレイアウトとグラフ表示を利用できます。
- **データ分析ページ**: スコア推移、対戦データ、キャラクターデータなどをより深く確認できます。
- **相手の傾向提示**: 相手の傾向をアプリ内で直接確認できます。
- **OBS 向け透過表示**: 配信画面へのオーバーレイに適しています。
- **レイアウト記憶**: ウィンドウ位置、サイズ、透明度、関連表示設定を自動で記憶します。
- **内蔵アップデーター**: 新しいバージョンをアプリ内で受け取れます。

## 使い方

1. `SF6 Scouter` をダウンロードして起動します。
2. ポップアップウィンドウで **CFN** にログインします。
3. デフォルトでは自分のアカウントを追跡し、必要に応じて別の **CFN ID** を入力して他プレイヤーも確認できます。
4. 用途に応じて **Mini** と **Pro** を切り替えます。
5. さらに詳しく見たい場合は **データ分析ページ** を開きます。
6. 透明度、レイアウト、表示方法を調整すると、設定は自動で保存されます。

## OBS 設定

1. OBS で **ウィンドウキャプチャ** を追加します。
2. `SF6 Scouter` ウィンドウを選択します。
3. キャプチャ方式は **Windows 10 (1903 以降)** を使用します。
4. 必要に応じて、アプリ内で透明度や表示設定を調整します。
